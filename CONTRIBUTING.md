# Contributing to Apocrypha

Apocrypha is an independent, Linux-first fork — small team, fast-moving.

- **Bugs and game requests**: use the [issue templates](https://github.com/Jeagermeister/Apocrypha/issues/new/choose). Logs live at `~/.local/state/Apocrypha/Logs` (Linux) or `%LOCALAPPDATA%\Apocrypha\Logs` (Windows).
- **Pull requests**: welcome. Build with the .NET 9 SDK (`dotnet build`), run the tests (`dotnet test`), and keep changes focused. CI runs the full suite on Linux and Windows for every PR.
- **Questions and ideas**: [Discussions](https://github.com/Jeagermeister/Apocrypha/discussions).
